# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行眼镜检测
- 支持选择不同的检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 提供自定义置信度和IOU阈值设置
- 可选择显示标签和原图

## 适用场景
- 眼镜产品分类与检测
- 视频监控中的眼镜识别
- 批量图片处理与分析
- 实时摄像头目标检测

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Python 3.9开发环境
- 集成OpenCV、PyQt5等技术
- 提供1500张眼镜标注数据集
- 包含完整源码、训练好的模型及运行教程
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
